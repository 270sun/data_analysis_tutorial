 * Day 3: 介绍预测性分析的方法和要点，以及动手实验和作业。

 * Day 5: 介绍关联规则挖掘的方法和要点，以及动手实验和作业。
 * Day 6: 介绍时间序列分析的方法和要点，以及动手实验和作业。

 * Day 9: 通过汽车之家的评论数据分析主要车型在市场上的情况。
